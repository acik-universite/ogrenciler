# ogrenciler
"Öğrenciler" reposu, Acık Üniversite'nin mezunlarının ve aktif öğrencilerinin profesyonel profillerini sergileyebilecekleri bir platformdur. Bu repo, öğrencilerin CV'lerine ekleyebilecekleri kişisel ve akademik bilgilerini, projelerini ve becerilerini içeren sayfaları içerir. Amacımız, öğrencilerimizin ve mezunlarımızın profesyonel görünürlüğünü artırmak ve kariyer fırsatlarına erişimlerini kolaylaştırmaktır.


### Kendi Kaydınızı güncellemek istediğinizde yapmanız gerekenler: (Bu Proje altında görünmesi için)

1. Projeyi `git clone` ile bilgisayarınıza indirin.
2. `ogrenciler` klasörüne gidin.
3. Yeni bir branch oluşturun. (Örneğin: `git checkout -b alameddin_celik_temmuz24`)
4. Adınıza ait bir md dosyası varsa (örneğin: alameddin_celik.md) açın.
5. Dosyaya erişim bilgilerinizi, yaptığınız projeleri, aldığınız dersleri ekleyin/düzenleyin. 
6. Güncelleme sonrasında, herşeyin düzgün olduğundan emin olduktan sonra, değişiklikleri `git add alameddin_celik.md(sizin dosya isminiz.md)` ve `git commit -m "Alameddin Çelik güncellendi"` komutları ile kaydedin.
7. GitHub'a push edin. `git push origin alameddin_celik_temmuz24` (Branch isminiz ne ise onu yazın)
8. GitHub üzerinden pull request oluşturun ve Discord üzerinden review talebinde bulunun.
9. Pull request'iniz onaylandıktan sonra, değişiklikleriniz `main` branch'ine merge edilecektir.

### Başka birine referans mektubu eklemek istediğinizde yapmanız gerekenler: (Bu Proje altında görünmesi için)

1. Projeyi `git clone` ile bilgisayarınıza indirin.
2. `ogrenciler` klasörüne gidin.
3. Yeni bir branch oluşturun. (Örneğin: `git checkout -b referans_mektubu_<referans_olanin_adi>_<referans_olanin_soyadi>_<tarih>`)
4. Referans mektubu eklemek istediğiniz kişinin adı ve soyadı ile başlayan dosyayı açın. (Örneğin: alameddin_celik.md)
5. Dosyaya referans mektubunu ekleyin. (Örnek kısmı templatelerden alabilirsiniz)
6. Güncelleme sonrasında, herşeyin düzgün olduğundan emin olduktan sonra, değişiklikleri `git add alameddin_celik.md(referans mektubu eklenen dosya ismi)` ve `git commit -m "Referans mektubu eklendi"` komutları ile kaydedin.
7. GitHub'a push edin. `git push origin referans_mektubu_<referans_olanin_adi>_<referans_olanin_soyadi>_<tarih>` (Branch isminiz ne ise onu yazın)
8. GitHub üzerinden pull request oluşturun ve Discord üzerinden review talebinde bulunun.

### Yapılabilecek hatalar:
1. Dosya adlarında boşluk bırakmayın
2. Uzantıyı yanlış yazmayın (Örneğin: .md)
3. Branch isimlerini yanlış yazmayın (Örneğin: alameddin_celik_temmuz24 veya referans_mektubu_alameddin_celik_sevket_yilmaz_2024)
4. Pull request oluştururken, hedef branch'in `main` olduğundan emin olun.
5. Reviewer eklerken, doğru kişileri eklediğinizden emin olun.
